PhaseFinder v0.1.0

PhaseFinder is a browser-based tool for inspecting and plotting flow cytometry .fcs files. It is designed as a lightweight, local-first application, where all data and analysis are performed on the user’s machine without uploading user data to any server.

🔗 Access the site: https://PhaseFinder.LatentLens.org
